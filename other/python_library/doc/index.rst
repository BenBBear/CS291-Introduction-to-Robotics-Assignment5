Welcome to Python iRoomba Create Interface's documentation!
===========================================================
This project contains two main modules. PICI ("peachy") is the high-level interface in Python to interact with the iRobot Create via PySerial.  PICS ("peaches") is the simulation environment that can take the place of a Create robot for development and testing.
